## ✨ What is this

**McTokenConverter** is an simple token converter for Minecraft players.

Drop in a Refresh Token (or a Microsoft login cookie) and it converts it into a usable Access Token

> [!WARNING]
> For learning and personal account management only. Do not use it for anything that violates the Minecraft / Mojang / Microsoft Terms of Service. Use at your own risk.
